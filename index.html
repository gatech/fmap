FMAP

Feature Matching Across Platforms

Download this project as a .zip file Download this project as a tar.gz file

Welcome to the FMAP repository.

FMAP is a technique and the name of our tool to perform application feature matching of web applications across desktop and mobile platforms. Currently our paper is under submission and we have created this website for reviewers. We will release more details about the paper soon!

Artifacts

FMAP is organized on github as follows.

  • http-trace: The trace extraction tool implemented as a extension to the Google Chrome platform
  • traces: traces collected by http-trace
  • fmap: Code for performing the feature matching
